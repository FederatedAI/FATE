## 底座层接口定义

### 1. 组件信息

#### 1.1 运行信息上报

- 功能描述：组件运行结束后将结果上报给本方服务器
- 接口地址：/local/task/result/report
- 调用方式：POST
- 请求参数：状态，失败信息等

- 响应参数：待定




### 2. 组件输出

#### 2.1 模型

- 功能描述：保存组件输出模型
- 接口地址：/client/task/output/model/save
- 调用方式：POST
- 请求参数：待定
- 响应参数：待定

#### 2.2 指标

- 功能描述：保存组件输出指标
- 接口地址：/local/task/output/summary/save
- 调用方式：POST
- 请求参数：待定
- 响应参数：待定
